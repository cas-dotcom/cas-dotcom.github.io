DNS & Email MX Lookup Lab

Terminal-based investigation using dig on Ubuntu to analyze UCF mail records

Overview

For this networking lab I captured and analyzed DNS traffic while resolving mail records for UCF. Using dig, I retrieved the MX record set and correlated A records for the mail exchangers. I verified response codes, authorities, and additional sections to confirm correct routing to ucfsmtp.mail.ucf.edu.

What I built / validated

  • Issued dig MX and dig A lookups against university mail hosts
  • Interpreted ANSWER, AUTHORITY, and ADDITIONAL sections
  • Checked TTLs, response sizes, and name server behavior
  • Documented findings with screenshots and concise notes

Skills learned

  • DNS fundamentals (A, MX, NS; recursion vs. iteration)
  • Linux CLI tooling: dig, terminal navigation
  • Troubleshooting email delivery paths at the DNS layer
  • Clear documentation with reproducible commands
