Overview

Nile Dot Com is a Java Swing application that simulates an e‑commerce storefront. Users can search inventory, add items to a cart, and generate receipts. The app reads product data from CSV files and writes transactional logs, demonstrating file I/O and event-driven programming.

Skills Learned

  • Java Swing GUI and event-driven programming
  • CSV parsing and file I/O
  • State management for cart, items, and totals
  • Usability considerations and error handling
