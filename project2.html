AWS Auto Scaling Web Server Application

Highly available EC2 fleet with a Launch Template and Auto Scaling Group

Overview

I deployed a web tier using an EC2 Launch Template and an Auto Scaling Group in us‑east‑1. The group maintained capacity by launching instances across subnets and scaling to meet policy targets. Health checks ensured only healthy instances served traffic.

Architecture highlights

  • EC2 Launch Template for immutable configuration (AMI, instance type, SGs)
  • Auto Scaling Group with min/desired/max capacity and activity history
  • Target group integration for load-balanced HTTP service (lab setup)
  • Tags and IAM roles for easier operations and logging

Skills learned

  • AWS EC2, Launch Templates, and Auto Scaling Groups
  • Target groups / health checks basics
  • VPC networking, subnets, and security group rules
  • Operational hygiene: tagging, IAM roles, and activity audits
