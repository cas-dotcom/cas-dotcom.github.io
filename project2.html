CNT 4603 SysAdmin Project: AWS Auto Scaling Web App

EC2 Launch Template + Auto Scaling Group for a highly available web tier

Overview

I deployed a web tier in us-east-1 using EC2 Launch Templates and an Auto Scaling Group with desired capacity, health checks, and tagging. The group added instances to maintain capacity, as shown in the scaling activity.

Skills & Outcomes (per rubric)

  • AWS EC2, Launch Templates, and Auto Scaling Groups
  • Target groups and health checks basics
  • VPC networking and security groups
  • Operational tagging, IAM roles, and activity auditing
